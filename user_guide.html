Presentation

BIAFLOWS online instance can be accessed at https://biaflows.neubias.org. It is possible to browse the content in read only mode from the guest account (username: guest; password: guest). Workflows cannot be launched from the guest account but all results from past workflow runs can be browsed. The platform has been tested for Chrome, Chromium, Safari and Firefox

Problems

List

This section brings an overview of all BIA problems available in BIAFLOWS. Each problem illustrates a common BIA problem and gathers a set of annotated images and compatible workflows to process them. A BIA problem class (e.g. Object Segmentation), is associated to every problem; BIAFLOWS currently supports 8 problem classes.

Problem class specify a format for ground truth annotations (identical to expected workflow output), as well as the benchmark metrics to be evaluated on the annotated images of a given Problem, see the detailed section (5 and 7) on metrics in Supplementary material.

Clicking on a problem redirects to the the problem images.

Images

This table lists all the images for the current Problem, where details can be displayed for each image:

Images can be filtered by name by typing a keyword in the search bar and visualized by clicking on their thumbnails.

In case ground truth annotations are stored as image masks (this depends on the problem class), the masks can be displayed by adapting filters (click on "Show filters" then select Type > Groundtruth). If ground truth annotations are in a different format (e.g. SWC, text file), they can be accessed through the Attached Files in the Image description.

Note: To visualize binary masks, it is recommended to set Gamma to 0.1 (see image viewer section).

Finally, all images can be downloaded through the button in the Actions section of the Image description.

Image Viewer

Navigation:
Use mouse scroll to zoom in/out, click + drag to move around. For multidimensional images, C-Z-T slicers automatically appears (when applicable) to navigate through the slices.

Layers:
To overlay an annotation layer, click on in the right toolbar. Annotation layers are either human ground-truths, or generated from the results of a workflow run (a.k.a. job). Several layers can be added hence overlaid on the original image, and they can be hidden by unselecting the checkbox under the symbol, or removed using . In addition, Layer opacity can be adjusted with the bottom slider. If you do not know which annotation layer to display, click on . Favorite jobs are workflow runs validated by their maintainer as representative results.

Multiple viewers:
It is also possible to have multiple viewers opened at once and synchronize them (e.g. 4 synchronized views where top left: original image, others: workflow results overlays):
This feature can be used to open the same image several times and compare different workflow results through the display of corresponding annotation layers. To do so, select an image from the upper left drop down menu and click on the big blue button (bottom right corner), select a workflow in the annotation layers , then use (in the right toolbar) to synchronize the views.

The current image can be downloaded by clicking on the image information panel in the right toolbar, then on (next to More information). For 2D images, it is also possible to switch to previous/next image using corresponding buttons ( / ) from this panel.

Finally, clicking on an annotation in the viewer will display some information (e.g. annotation area and perimeter), providing a link to the description of the workflow that generated it, or center the view based on this annotation.

Image adjustments:
Contrast and gamma correction can be adjusted from the panel (right toolbar). The image can also be video inverted from this panel.
